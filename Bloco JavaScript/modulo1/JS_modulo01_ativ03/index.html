EDITOR DE CÓDIGO VSCODE

Escolhi o VScode pelo fato dele possuir uma interface simples e ser bem mais leve do que usualmente as IDEs são.Além de ser mais flexível com relação ao que instalar (Eu posso optar por instalar somente as extensões ou linguagens que desejo)

Em comparação com outros editores de código, o VSCode se mostra mais completo, mais famoso ( o que faz com que as comunidades estejam sempre aprimorando as ferramentas e extensões para esse editor de códigos). Outras características positivas do VSCode é o fato dele ser um software livre de código aberto, ter controle git incorporado e possuir suporte para depuração
